# 项目地图

项目是什么
- AI Marketing System 下的 CRM 子模块。
- 目标是线索管理、AI 评估、轻咨询与权限隔离。
- 未来可拆分为 SaaS，并与新媒体营销系统联动。

项目在做什么
- 建立 CRM 核心闭环。
- 预留 AI 评估与新媒体营销子系统接口。
- 设计文档已完成，进入 MVP 阶段。

模块与边界
- CRM：线索、客户、商机、跟进、轻咨询。
- AI 评估：独立服务，手动触发。
- 新媒体营销：独立系统，后续打通。
- just right 官网：获客入口，表单入 CRM。

当前阶段
- 设计阶段已完成。
- 待进入 MVP 开发。
