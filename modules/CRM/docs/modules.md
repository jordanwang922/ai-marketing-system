# 模块规划

基础模块
- 组织与品牌管理
- 用户与角色管理
- 权限与隔离规则
- 审计与操作日志

CRM 业务模块
- 线索池（Leads）
- 联系人与公司（Contacts/Companies）
- 商机与阶段（Deals/Pipeline）
- 跟进与任务（Activities/Tasks）
- 轻咨询（Consulting）

AI 模块
- 线索调研与评分
- 调研报告生成
- 评分规则与可解释输出

集成模块
- 官网表单入库
- 邮件服务
- Webhook 与 API 扩展


新媒体营销系统（独立子模块）
- 市场与平台情报
- 关键词与竞品洞察
- 内容生成与多平台适配
- 数据反馈与表现分析
- SaaS 化与多租户隔离
- 客户案例库与案例驱动内容生成
