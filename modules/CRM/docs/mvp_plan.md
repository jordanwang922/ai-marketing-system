# MVP 交付规划

目标
- 在最短时间内打通 CRM 核心闭环：线索进入 → 去重 → 分配 → 跟进 → 轻咨询 → 邮件交付。
- 支持中英 UI 与 H5 自适配。

MVP 范围（必须有）
1. 线索池
- 手工录入线索
- 官网表单入库
- 公司名去重与合并
- 线索状态流转

2. 分配与权限
- 主管可见团队线索
- 销售仅见本人线索
- 主管手工分配线索

3. AI 评估（手动触发）
- 按公司名发起评估
- 回写评分与摘要

4. 轻咨询
- 创建轻咨询单
- 指派专家
- 专家回传语音/文本
- 系统整理成邮件并发送给客户

5. 通知
- 站内信与邮件通知（最小可用）

6. 多语言与移动端
- 中英 UI
- H5 自适配覆盖核心流程

MVP 范围外（后续）
- 自动分配规则
- 复杂数据安全策略
- 新媒体营销系统打通
- 专家开放平台

交付顺序建议
1. 数据模型 + 权限隔离
2. 线索池 CRUD + 去重合并
3. 表单入库 + 手工分配
4. AI 评估触发 + 评分回写
5. 轻咨询流程 + 邮件交付
6. 通知 + 多语言 + H5 适配
