# 电脑端 + 手机端 H5 适配说明

目标
- 电脑端完整功能。
- 手机端 H5 主要覆盖高频核心流程：线索录入、线索列表、客户详情、跟进记录、轻咨询处理。
- 视觉风格与 justright-calendar 一致，保持精致与统一。

适配策略
1. 响应式优先
- 使用 Tailwind 的响应断点（sm/md/lg/xl）。
- 采用移动优先布局，桌面端增强。

2. 布局原则
- 桌面端：侧边栏 + 内容区的双栏结构。
- 移动端：底部导航 + 轻量顶部栏，侧边栏折叠为抽屉。

3. 交互规则
- 列表在移动端采用卡片式布局。
- 表格在移动端转为卡片或折叠展示。
- 表单控件放大点击区域，避免误触。

4. 导航规范
- 移动端导航固定 4-5 个核心入口：线索、客户、商机、跟进、我的。
- 桌面端完整导航保留。

5. 表单体验
- 分步骤表单（stepper）适配小屏。
- 长表单按分区折叠。

6. 视觉一致性
- 复用 justright-calendar 的 Tailwind 变量与组件风格。
- 保持卡片圆角、阴影、间距体系一致。
- 移动端强调“精致感”：大留白、清晰层级、少而精的交互按钮。

7. 性能与可用性
- 首屏关键内容优先渲染。
- 图片与图表懒加载。
