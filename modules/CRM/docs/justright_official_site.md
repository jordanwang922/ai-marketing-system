# Just Right 官网内容结构与表单流程

目标
- 作为品牌展示与获客入口。
- 展示服务能力与成功案例。
- 通过表单进入 CRM 线索池。
- 中英文双语，默认中文，可手动切换英文。
- 内容以静态为主，半年到一年内不频繁修改。

内容结构建议
1. 首页 Hero
- 核心价值主张
- 关键数据与成果
- 主要 CTA（立即咨询/提交需求）

2. 公司介绍
- 公司定位与使命
- 服务覆盖国家与行业

3. 服务能力
- 出海服务类型
- 轻咨询与深度服务

4. 成功案例
- 图文案例卡片
- 可筛选（行业/地区/服务类型）

5. 客户证言
- 可选视频/文字

6. 线索表单
- 基础信息（公司名、联系人、邮箱、国家、需求）
- 提交后进入 CRM 线索池

7. 联系方式
- 邮箱/微信/电话

表单流程
1. 用户提交表单
2. 表单数据入 CRM 线索池
3. 进入共享线索池，待分配

视觉方向
- 保持 justright-calendar 风格：现代、干净、细节精致
- 强动效（但不干扰阅读）
- 深浅背景对比，突出 CTA

技术栈建议（待评估）
- 初期建议与 CRM 同栈：Vite + React + Tailwind
- 若需极致 SEO 与长期稳定发布，可考虑 Next.js 静态导出
