# 权限与品牌隔离

角色分层
- 超级管理员
- 品牌管理员
- 主管
- 销售
- 专家

可见性规则
- 用户只属于一个品牌，不可跨品牌。
- 主管可见其团队所有客户与线索。
- 销售只能看自己负责的客户与线索。
- 团队成员之间互不可见客户资料。
- 主管之间互不可见其他主管与团队资料。
- 不同品牌之间严格隔离，不可跨品牌查看。

实现要求
- 数据层强制 brandId 过滤。
- API 层强制权限校验。
- 前端仅展示授权范围内数据。
- 支持线索共享/转移：\n+  - 共享线索入口：官网表单。\n+  - 分配方式：自动规则或人工分配（两种均支持）。

审计要求
- 访问与编辑客户信息必须记录。
- 敏感操作记录操作者与时间。
