# AI 子系统规划

AI 评估系统
定位
- 根据公司名称进行调研与评分。
- 输出可解释结论，建议是否继续跟进。

能力范围
- 全网信息抓取与摘要。
- 可信度评估与风险提示。
- 评分规则可配置（0-100 分）。

集成方式
- CRM 通过 API 请求 AI 评估。
- 结果回写至 Lead 与 AIEvaluation。
- 评估触发：由用户点击触发，不自动评估。

新媒体营销系统
定位
- 独立子系统，目标是面向客户 SaaS 化。
- 自动发现行业关键词与爆款内容。
- 生成多平台内容与脚本。
 - 支持客户案例库，并基于案例生成营销软文与故事。

能力范围
- 关键词与竞品分析
- 内容选题与脚本生成
- 多平台分发模板与风格控制
 - 客户案例库（图文/文字）
 - 案例驱动的营销故事生成

CRM 打通方向
- 初期不打通。
- 后续以客户线索为输入，驱动营销内容与触达策略。

专家服务平台（后续）
定位
- 开放式专家供需平台，支持专家入驻与撮合服务。
- 可与轻咨询流程打通。
